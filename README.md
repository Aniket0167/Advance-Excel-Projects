 Overview

The Road Accident Dashboard is an interactive Excel-based analytics solution designed to analyze and visualize road accident casualty data across multiple dimensions such as severity, vehicle type, road conditions, location, and time.
The dashboard provides a clear, data-driven view of accident trends, helping stakeholders understand risk factors and identify areas for safety improvement.

This project demonstrates strong skills in Excel data modeling, dashboard design, KPI creation, and visual storytelling.
